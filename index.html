HTML

Esqueleto basico de um HTML:


                <!DOCTYPE html>
                <html lang="pt-BR">
                <head>
                <meta charset="UTF-8">
                <title>Título da Página</title>
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <link rel="icon" href="favicon.ico">
                <style>
                /* CSS interno */
                </style>
                <!-- Ou link para CSS externo -->
                <!-- <link rel="stylesheet" href="estilos.css"> -->
                <!-- Outros metadados -->
                </head>
                <body>
                <header>
                <!-- Cabeçalho -->
                </header>
                <main>
                <!-- Conteúdo principal -->
                </main>
                <footer>
                <!-- Rodapé -->
                </footer>
                <!-- Scripts JS geralmente vão aqui, antes de fechar o body -->
                <script src="script.js"></script>
                </body>
                </html>

Tags com fechamento:

  • <p> (Parágrafo)
  • <html> (Início e fim de todo HTML)
  • <head> (Conjunto de instruções pra página como um todo)
  • <body> (Conteúdo do corpo da página)
  • <h1-6> (Títulos e seus subtítulos)
  • <a> (Links)
  • <ul-ol> (Listas não ordenada e ordenada, respectivamente)
  • <li> (Conteúdo dessas listas)
  • <div> (Contêiner de elementos que está dentro de <section> geralmente)
  • <section> (Contêiner de elementos que contem <div> dentro geralmente)
  • <header> (Cabeçalho que fica no <body>)
  • <nav> (Sumário que fica no <header> que fica dentro do <body>)
  • <span> (Contêiner para pequenos textos ou pequenas partes de texto)
  • <main> (Conteúdo principal e único da página)
  • <footer> (Rodapé da página)
  • <button> (Botão clicável)
  • <form> (Criar formulários)
  • <figure> (Geralmente se coloca img e figcaption que é a legenda da imagem, uma forma mais profissional de inserir imagens)
  • <textarea> (Para áreas de textos maiores)
  • <label> (Serve para criar um conteiner e colocar um input dentro, pra acessibilidade e organização)
  • <strong> (Negrito de forma mais facil e semântica)
  • <em> (Itálico de forma mais facil e semântica)

Tags sem fechamento:

  • <br> (Quebra de linha)
  • <hr> (Usada para criar uma linha horizontal para separar o conteúdo)
  • <img> (Inserir imagem numa página, geralmente usada dentro do bloco semântico figure)
  • <meta> (Regras da página, informações sobre a página, como o conjunto de caracteres)
  • <link> (Usada para vincular a página a outros recursos externos, com o css e o js)
  • <input> (Permite o usuário inserir dados na página)

Atributos:

  • placeholder="..." (Texto em área de digitação antes do usuário digitar)
  • class="..." (Uma forma de referenciar o local no css)
  • id="..." (Outra forma de referenciar no css)
  • style="..." (Permite adicionar regras de CSS diretamente na tag, para estilos rápidos e específicos)
  • href="..." (Usado dentro da tag <a> para inserir links)
  • src="..." (Caminho ou link para uma certa imagem, dentro de img ou da tag figure)
  • alt="..." (Legenda na imagem caso ela não tenha exito em carregar)
  • type="..." (Usado dentro do input para dizer o tipo de campo que o usuário vai preencher, texto, número, checkbox, radio, etc)
  • name="..." (Adiciona um nome ao campo do formulário, para enviar corretamente ao servidor, checkbox sempre marca mais de um, radio não, somente quando tiver name diferente)
  • minlenght="..." (Quantidade minima a se preencher)
  • maxlenght="..." (QUantidade máxima a se preencher)
